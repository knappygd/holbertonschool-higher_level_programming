 Test 1-my_list.py
===============================

This library is very simple, since it only has one class called MyList.


::

    >>> MyList = __import__('1-my_list').MyList
    >>> my_list = MyList()
    >>> my_list.append(1)
    >>> my_list.append(4)
    >>> my_list.append(2)
    >>> my_list.append(3)
    >>> my_list.append(5)
    >>> print(my_list)
    [1, 4, 2, 3, 5]
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5]
    >>> print(my_list)
    [1, 4, 2, 3, 5]

::

    >>> my_list2 = MyList()
    >>> print(my_list2)
    []
    >>> my_list2.print_sorted()
    []
    >>> print(my_list2)
    []

::

    >>> my_list3 = MyList([-1, -2, -4, 5])
    >>> print(my_list3)
    [-1, -2, -4, 5]
    >>> my_list3.print_sorted()
    [-4, -2, -1, 5]
    >>> print(my_list3)
    [-1, -2, -4, 5]

::

    >>> my_list4 = MyList([5, -1, -4, -2, -4, 5])
    >>> print(my_list4)
    [5, -1, -4, -2, -4, 5]
    >>> my_list4.print_sorted()
    [-4, -4, -2, -1, 5, 5]
    >>> print(my_list4)
    [5, -1, -4, -2, -4, 5]
